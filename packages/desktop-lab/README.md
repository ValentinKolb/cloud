# Desktop Lab

This directory is retained as an inactive experiment. It is not part of the root Bun workspace, Docker builds, release artifacts, or standard verification, and a fresh clone does not provision its native Electrobun toolchain.

Use the shared APIs under `packages/cloud/src/desktop` for maintained desktop SDK work. Do not use this experiment as a supported starter application without first promoting it into the workspace and defining a maintained install, build, and release path.
